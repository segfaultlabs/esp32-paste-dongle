HID Mode

Keyboard + Mouse — normal mode: paste text, run macros, jiggle mouse.
Mouse Only — jiggler use only. No keyboard interface is registered in the USB descriptor. The device appears as a pure mouse — significantly harder to detect by monitoring software. Paste and macro features are disabled in this mode.

Switching modes requires uploading a different firmware binary. Build with:
pio run -e esp32s3-usb → Keyboard + Mouse
pio run -e esp32s3-mouse → Mouse Only
Then upload the .bin via the OTA section below.

Keyboard Layout

Must match your host OS keyboard setting (System Settings → Keyboard). Wrong layout = wrong characters.

HID Identity

One identity covers the whole device — both the keyboard and mouse interfaces share the same VID/PID and name. The host sees one USB device with two capabilities.

Pick a keyboard preset when pasting is your main use. Pick a mouse-forward preset when using only the jiggler — the host expects mouse movement but will also see a keyboard interface attached (uncommon but harmless). Requires reboot to apply.
