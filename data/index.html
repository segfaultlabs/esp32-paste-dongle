Human simulation

Keeps your session alive on applications that track keyboard activity separately from mouse movement (e.g. some VPNs, remote desktop clients, meeting tools). The dongle types random words and erases them in a loop — it looks like someone is at the keyboard.

How fast the simulation types each character. Slower speeds look more human; faster speeds are less noticeable but may look odd if the cursor is visible. 100–150 ms/char is the natural human range.

Mouse jiggler

Moves the mouse cursor periodically to prevent screen lock, sleep, or idle timeouts. The Natural mode is specifically designed to be undetectable by anti-idle software.

Update UI (no reboot needed)

Upload a new index.html to the device filesystem. The page reloads immediately — no firmware reflash required. Useful when iterating on the UI.

Update firmware (OTA)

Build in VS Code (PlatformIO → Build), then pick .pio/build/esp32s3-usb/firmware.bin. The dongle writes to the standby OTA slot and reboots — your previous firmware is kept. The page reloads automatically after ~12 seconds.
